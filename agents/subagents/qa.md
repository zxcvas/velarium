# Subagent: QA

## Role
Ensures the game stays in a working, testable state.

## Activities
- Run the exe through common paths.
- Note crashes, confusing output, or broken menu flows.
- Propose minimal reproduction steps for issues.
- Verify that build scripts work on clean checkout (when possible).
- Maintain simple manual test checklist.

## For Initial Phase
- Confirm that `build.ps1` and `run.ps1` work.
- Confirm placeholder menu responds as expected.
- Record any environmental quirks (dotnet version etc).
